# Product Context: Seimas v.2

## Why this exists
Democratic transparency requires accessible data. The official sources are often fragmented or difficult to consume for average citizens. This project centralizes that data to enable accountability.

## Target Audience
- Civic tech developers and data scientists.
*   Journalists and transparency watchdogs.
- Curious citizens interested in Lithuanian politics.

## User Experience Goals
- **Data Integrity**: Ensure the linked data across different sources (Seimas vs. VRK) is accurate.
- **Up-to-date Information**: Provide a pipeline that can be run regularly to keep data fresh.
- **Extensibility**: Make it easy to add new data sources (e.g., more detailed asset declarations or interest disclosures).
